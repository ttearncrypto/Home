User Guide

How to Search for Tools

Use the main search bar at the top of the page to find tools by their name. Type your keywords and the list will update automatically.

You can also click on the "Popular tools" links to quickly search for common terms.

Filtering by Category

Above the tool listings, you'll find category tabs (e.g., "Domain Tools", "DNS Tools"). Click on a tab to view only the tools belonging to that category.

Click "All Tools" to see tools from all categories.

Use the arrow buttons on the sides of the tab bar to scroll if there are many categories.

Sorting Tools

Below the main search bar (or beside it on larger screens), there's a dropdown menu that allows you to sort the tools. You can sort by:

  • Default Order: The order in which tools are fetched from the source.
  • Name (A-Z or Z-A): Alphabetical or reverse alphabetical order.
  • Newest/Oldest First: Based on when the tool was added or published.

Sorting applies to the tools within the currently selected category or search results.

Using Category Box Icons

Each category box (e.g., "Domain Tools") has two icons in its header:

  • View All Icon: Clicking this will switch the main filter tabs to this specific category and scroll you to the top, effectively showing all tools under this category based on your current sort order.
  • Search Category Icon: Clicking this will pre-fill the main search bar with the category's name and focus the search bar, allowing you to quickly refine or confirm a search for tools related to that category.
